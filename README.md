# Плагин emoticons под замену в TinyMCE4

Данный плагин заменяет установленны `emoticons` в редакторе.

Плагин не использует изображения эмоций, а использует реальные символы эмоций, что позволяет подключить шрифт `Noto Color Emoji` или другой подобный в проект сайта.

Сам же плагин использует `Noto Color Emoji` для отображения символов при выборе.
